# Contributing to Agent OS

Agent OS is in **Milestone 0 — Foundation Specification**. During this phase:

- The repository is **public** for transparency and visibility
- Contributions are accepted via **Issues** (discussion, feedback, proposals)
- **PRs are not yet open** — the core specification must stabilize before implementation contributions

## How to Contribute

### Issues
- **Bug reports**: Found an error in a specification? Open an issue.
- **Feature requests**: Have an idea for a new capability or workflow? Open an issue.
- **Discussion**: Questions about the architecture or design? Open a discussion.

### RFC Process
Once the core specification stabilizes, the RFC process will open for community proposals. See [RFC-INDEX.md](docs/rfc/RFC-INDEX.md).

## Code of Conduct

Be respectful. This project aims to build infrastructure, not ego.
