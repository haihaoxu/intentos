## Process

1. When a significant architectural decision is made, create an ADR
2. Format: `ADR-XXXX-title-with-hyphens.md`
3. Include: Context, Decision, Consequences, Alternatives Considered
4. ADRs are immutable once accepted — corrections come via new ADRs
